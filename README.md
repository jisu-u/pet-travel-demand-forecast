## Project 1
Pet Travel Demand Forecast

Dataset:
Google Trends (5 years)

Keywords:
- pet airline
- pet boarding
- pet friendly hotel
- pet friendly cafe
- pet friednly restarant

Region:
South Korea

Goal:
Understand and predict demand for pet travel services.

Methods:
- Python
- Pandas
- Numpy
- Linear Regression
- Data Visualization (Matplotlib)


Result:
Search interest in pet travel services shows an increasing trend, suggesting potential growth in the pet travel market.


Future Plan

Expand the analysis with more detailed datasets to create personalized pet travel insights.

- Analyze seasonal demand patterns for pet travel services.
- Compare regional demand differences across cities and countries.
- Segment travel demand by pet type (dogs, cats, rabbits, hamsters, etc.).
- Use these insights to design personalized VIP pet travel packages and tailored services for different pet owners.
